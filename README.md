## Setup ‚Äî GitHub Pages + Custom Domain

### Step 1 ‚Äî Upload All Files
Upload all files above to your repository root in one batch.

### Step 2 ‚Äî Enable GitHub Pages
1. Repository **Settings** ‚Üí **Pages**
2. Source ‚Üí **main** branch ‚Üí **/ (root)**
3. Click **Save** ‚Üí wait 2-3 minutes

### Step 3 ‚Äî Connect flecto.shop Domain (Hostinger DNS)
1. Hostinger ‚Üí Domains ‚Üí flecto.shop ‚Üí DNS Zone
2. Delete existing A records for @
3. Add 4 A records:
```
Type: A  |  Name: @  |  Value: 185.199.108.153
Type: A  |  Name: @  |  Value: 185.199.109.153
Type: A  |  Name: @  |  Value: 185.199.110.153
Type: A  |  Name: @  |  Value: 185.199.111.153
```
4. Add CNAME:
```
Type: CNAME  |  Name: www  |  Value: byshak.github.io
```
5. Wait 10-30 minutes

### Step 4 ‚Äî Custom Domain in GitHub
1. Pages settings ‚Üí Custom domain ‚Üí `flecto.shop` ‚Üí Save
2. Tick **Enforce HTTPS**
